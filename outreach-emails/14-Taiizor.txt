Hi Taiizor,

Sweeper is a great concept — a cross-platform CLI cleanup tool in Rust with a focus on extensibility, performance, and safety. That's exactly the right design philosophy for this kind of tool.

I'm building X-MaC, an open source macOS (and now Linux) diagnostic and maintenance tool with a Rust core. It scans for disk usage, cache bloat, build artifacts, PATH conflicts, broken symlinks, missing shared libraries, environment mapping, and system maintenance. It also has a GNN model (CoreML) for filesystem anomaly detection, a SwiftUI frontend, and generates safe remediation scripts.

Since we're both building Rust-based cleanup tools with cross-platform ambitions, I think there's a lot we could learn from each other:

https://github.com/davidnichols-ops/X-MaC

Discussions are enabled. I'd welcome any feedback, collaboration, or idea-sharing.
